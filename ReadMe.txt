Solution Implemented with Algorithms - DFS, BFS, AStar, BestFirst
**********************************************************

Instruction to run:

1. DFS - Single Path:  Command line arguments are : DFS {maze file path}

2. BFS - Single Path:  Command line arguments are :  BFS {maze file path}

3. Astar - Single Path: Command line arguments are :  AStar {maze file path}

4. BestFirst - Single Path: Command line arguments are : BestFirst {maze file path}

5. All Paths - Only DFS is implimented accordingly: (Will not work for other algos)
Command line arguments are : DFS {maze file path} all

If you are using other tool, any alternative way to run a jar with arguments will work
